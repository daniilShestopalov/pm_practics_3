## Расчет итоговых показателей
# Use Case Points

UCP = UUCP × TCF × ECF = 133 × 1.115 × 0.71 = 105.3

# Определение коэффициента производительности

Анализ факторов окружения для определения коэффициента:

- Факторы E1-E6 с оценкой >3: 2 фактора (E2, E5)
- Факторы E7-E8 с оценкой <3: 2 фактора (E7, E8)

Общее количество критических факторов: 4

Согласно методике:

3-4 критических фактора → коэффициент = 28 часов/UCP

# Расчет трудозатрат

Трудозатраты = UCP × Коэффициент = 105.3 × 28 = 2,948 человеко-часов

# Пересчет в человеко-месяцы:

Рабочих часов в месяце: 176 (22 дня × 8 часов)

Трудозатраты = 2,948 ÷ 176 = 16.75 человеко-месяцев
